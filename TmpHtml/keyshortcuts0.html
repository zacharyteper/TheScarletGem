Key Shortcuts

Here is a complete list of keyboard shortcuts within the game:


  • Ctrl+P - print
  • Ctrl+E - view high scores
  • Ctrl+S - save
  • Ctrl+U - view About dialog
  • Ctrl+H - view Help dialog
  • Ctrl+Q - quit/exit
  • Ctrl+R - how to play/view instructions
